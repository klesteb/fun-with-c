=head1 NAME

room-dump - dump the room datastore

=head1 SYNOPSIS

room-dump [-?|-h|-H] 

=head1 DESCRIPTION

This program is used dump the room datastore. Extremely helpful when
you mess up. All output goes to stdout, errors go to stderr.

=head1 OPTIONS AND ARGUMENTS

=over 4

=item B<-?|h|H>

Output simple help text.

=back

=head1 EXIT CODES

 0 - success
 1 - failure
